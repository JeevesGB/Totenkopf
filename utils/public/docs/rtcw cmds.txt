Core Cheat Commands
Command	Effect
/god	God mode
/noclip	Fly through walls
/notarget	Enemies ignore you
/nofatigue	Infinite sprint
/give all	All weapons/ammo/items
/give ammo	Full ammo
/give armor	Max armor
/give armor 999	Specific armor amount
/give health	Full health
/give health 999	Specific health amount
/give stamina	Restore stamina
/kill	Suicide
/cg_FOV 90	Change FOV
/cg_thirdperson 1	Third-person camera
/cg_thirdperson 0	Return to first-person
/quit	Exit game
/mapname	Current map name
/serverinfo	Server info
/cmdlist	List all commands
/dir maps	List maps
/reconnect	Reconnect to server
/screenshot	Take screenshot
